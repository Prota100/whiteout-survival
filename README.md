# 🏔️ 화이트아웃 서바이벌 v2.1

극한의 설원에서 살아남아라! 1시간 생존 웹 게임.

## 🎮 플레이

**[▶️ 지금 플레이하기](https://prota100.github.io/whiteout-survival/)**

## ✨ 주요 기능

- **5가지 클래스**: 전사, 마법사, 서바이버, 샤먼, 헌터
- **40+ 업그레이드**: 뱀서라이크 스타일 카드 선택 시스템
- **장비 시스템**: 등급별 드롭 (일반~전설)
- **보스 러시**: Act별 보스 + 히든 보스
- **한파 시스템**: 주기적 블리자드, 화덕 근처에서 생존
- **건설 & 제작**: 화덕, 벽, 함정, 무기 제작
- **NPC 고용**: 동료 채용으로 전투력 강화
- **스킨 & 업적**: 수집 요소

## 📁 프로젝트 구조

```
src/
├── data/       # 게임 데이터 (업그레이드, 클래스, 장비 등)
├── systems/    # 오디오, 세이브, 메타, 시너지 등 시스템
├── scenes/     # Phaser 씬 (Title, Boot, Game)
├── utils/      # 헬퍼 함수
└── main.js     # Phaser 설정 & 실행
```
